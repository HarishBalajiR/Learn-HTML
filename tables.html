Tables in HTML

What is a table?

  • A table contains rows and columns to store data.

  • Data can be better analyzed using tables

Tags

  • Table Tag: Declares the start of a table
  • border="1": Shows a border between the rows and columns.
  • tr: Short for Table Row(tr), denotes the current row
  • th: Short for Table Header(th), use only for table headings, makes them centered and look good.
  • td: Short for Table Data(td), denotes the data entered in the table.
  • thead: Short for Table Head(thead), enclose th tags in this because it might look broken when adding CSS.
  • tbody: Short for Table Body(tbody),enclose td tags in this tag.
  • rowspan,colspan: Refer below
